Trading signal: 10Y Bond Yield

If 10Y yield rises → expect TLT to drop → short/sell TLT.

If 10Y yield falls → expect TLT to rise → buy TLT.

We can see that the sharpe ratio of the first strategy is far more better than the momentum strategy and the Z-score strategy. So I chose the short long MA strategy as my main strategy and I want to improve the strategy to make it more profitable. Strategy 3 performs far more better than Strategy 1, with higher sharpe ratio and annual return. In this way, we can see that if we can know today’s price ahead, then we can definitely improve the strategy’s performance. So I canme up with the idea to predict the price of t based on history date up to t-1 using LSTM. And I tried training the LSTM model only once on the train data set and use this model to predict the test set, which is exactly what Strategy 4 is doing.

 

 

 

strategy 4: directly use LSTM to predict next day’s yield:

 

 

And we can see that the strategy can’t precisely predict the price we want, so I am thinking of implementing a rolling forecast approach — continuously retraining the model on the most recent data and generating updated predictions at each step — to adapt to changing market conditions and reduce lag between yield movements and TLT price reactions. So here comes Strategy 5.
